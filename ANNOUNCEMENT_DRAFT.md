# Announcement Draft

あなたの、絶対に残したい1秒はなんですか？

パラパラQR という試作を公開しました。

これは、短い音声や動画を QR コード群として紙や画面に焼き付け、別の端末で復元するためのプロジェクトです。

目指しているのは、便利な共有ではなく、
**ずっと、のこせるデジタル**。

サービスやサーバーに預けるのではなく、
**紙が、永遠のサーバーになる**
ような保存の仕組みをつくれないか、という試みです。

現時点では `v0.1` の public preview ですが、すでに次のことを確認しています。

- 単一QR / 複数QRで短い音声を生成できる
- GitHub Pages 上の decoder から iPhone で読み取って復元できる
- ローカル保存まで持っていける

まだ読み取りやすさや UX は改善途中です。
でも「紙へデータを焼き付ける」というコンセプト自体は、もう動いています。
